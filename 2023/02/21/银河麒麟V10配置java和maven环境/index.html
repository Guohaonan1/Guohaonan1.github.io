配置java环境

架构为intel

系统自带

如果开发java应用，经常需要配置JAVA_HOME路径，如果是通过yum安装的jdk（一般系统会自带open-jdk），需要定位JDK安装路径

至此，确定java的安装目录为： /usr/lib/jvm/java-1.8.0-openjdk-1.8.0.xx.x86_64

自己安装

安装openjdk1.8，Oracle JDK由Oracle Corporation完全开发，而OpenJDK由Oracle，OpenJDK和Java Community开发。==OpenJDK是开源的==。

openjdk官网下载：https://jdk.java.net/java-se-ri/8-MR4

java的安装目录为： /usr/local/java-se-8u42-ri

配置jdk环境变量

JAVA_HOME的值就是系统自带的或自行安装的jdk的位置
